# 冲压工艺教导插入流程

## 概述

本文档详细说明了在冲压工艺中，从选择工艺类型到将冲压模块参数插入教导程序的完整流程。

## 流程架构图

```
用户操作
  ↓
工艺类型选择 (ProductSettings.qml)
  ↓
加载工艺设置页面 (PunchSettingPage.qml)
  ↓
教导程序页面 (ProgramPage.qml)
  ↓
程序流程页面 (ProgramFlowPage.qml)
  ↓
工艺模块菜单 (ProcessMenu.qml)
  ↓
冲压工艺模块 (StampingTechnology.qml)
  ↓
生成指令对象 (createActionObjects)
  ↓
插入教导程序 (insertActionToList)
```

### 6. 冲压工艺模块（核心）

**文件**: `qml/App_HAMOUI/teach/StampingTechnology.qml`

这是生成取片指令对象的核心模块：

```qml
ExtentActionEditorBase {
    id: stampingSystem
    
    // 创建指令对象数组
    function createActionObjects() {
        var ret = [];
        
        // 获取用户设置的参数
        var action = pointEdit.getAction();        // 动作类型（直线、圆弧等）
        var points = pointEdit.getPoints();        // 点位数据
        var flag = pointEdit.getStamping();        // 冲压类型（0-14）
        
        if(action < 0 || points.length < 1) return ret;
        
        var speed = pointEdit.getSpeed();          // 速度
        var delay = pointEdit.getDelay();          // 延时
        var smooth = pointEdit.getSmooth();        // 平滑度
        var mask = pointEdit.getCKStatus();        // 轴掩码
        var toolCoord = pointEdit.getToolCoordBind(); // 工具坐标
        
        // 提取点位坐标
        var point1, point2, point3, point4, point5, point6;  // 第一个点的6轴坐标
        var point7, point8, point9, point10, point11, point12; // 第二个点的6轴坐标
        var m7, m8, m7_p, m8_p;  // 第7、8轴坐标
        
        // 从points数组中提取坐标值
        if(points.length === 1) {
            info1 = points[0].pos;
            for(i in info1) {
                if(i=="m0") point1 = info1[i];
                if(i=="m1") point2 = info1[i];
                // ... 提取所有轴坐标
            }
        } else if(points.length === 2) {
            // 提取两个点的坐标（用于圆弧等）
            info1 = points[0].pos;
            info2 = points[1].pos;
            // ...
        }
        
        // 根据冲压类型生成不同的指令对象
        if(flag == 1) {
            // 取片指令（action 43）
            tmp = {
                "action": 43,                    // 指令类型：输出+路径
                "PathAction": action,            // 路径动作类型
                "speed": speed,                  // 速度
                "delay": delay,                  // 延时
                "OutputAction": 201,             // 输出动作
                "intervalType": false,           // 间隔类型
                "isBindingCount": false,         // 是否绑定计数器
                "pointStatus": true,             // 点位状态
                "point": 1,                      // 冲压类型标识
                "type": 5,                       // 类型
                "counterID": "0",                // 计数器ID
                "cnt": "0",                      // 计数
                "delay2": "1.0",                 // 第二延时
                "point1": point1, "point2": point2, "point3": point3,
                "point4": point4, "point5": point5, "point6": point6,
                "point7": point7, "point8": point8, "point9": point9,
                "point10": point10, "point11": point11, "point12": point12,
                "smooth": smooth,                // 平滑度
                "m7": m7, "m8": m8,             // 第7、8轴
                "m7_p": m7_p, "m8_p": m8_p,     // 第二点的第7、8轴
                "mask": mask,                    // 轴掩码
                "points": points,                // 原始点位数组
                "toolCoord": toolCoord,          // 工具坐标
                "quotePoint1": 0,                // 引用点1
                "quotePoint2": 0                 // 引用点2
            };
        } else if(flag == 3 || flag == 10 || flag == 11 || flag == 12 || flag == 13 || flag == 14) {
            // 其他取片类型（间隔取片等）
            tmp = {
                "action": 43,
                "intervalType": true,            // 间隔类型为true
                "delay2": "0.0",                 // 第二延时为0
                // ... 其他参数同上
            };
        } else if(flag == 0 || flag == 2) {
            // 等待类型的取片（action 46）
            tmp = {
                "action": 46,                    // 指令类型：等待+路径
                "PathAction": action,
                "speed": speed,
                "delay": delay,
                "AwaitAction": 100,              // 等待动作
                "type": 5,
                "point": flag,
                "pointStatus": 2,
                "limit": "6000000",              // 超时限制
                "isUnlimit": false,
                // ... 点位参数
            };
        } else {
            // 其他类型（action 49）
            tmp = {
                "action": 49,
                "PathAction": action,
                "speed": speed,
                "delay": delay,
                "point": flag,
                // ... 点位参数
            };
        }
        
        // 生成自定义动作并添加到返回数组
        ret.push(Teach.generateCustomAction(tmp));
        return ret;
    }
    
    // 点位编辑器
    PointEdit {
        id: pointEdit
        isStampingMode: true  // 冲压模式
    }
}
```

function onInsertTriggered() {
    // 调用动作编辑器框架的createActionObjects()
    // 这会调用到ProcessMenu.createActionObjects()
    // 最终调用到StampingTechnology.createActionObjects()
    var actionObjects = actionEditorFrame.item.createActionObjects();
    
    // 检查是否有有效的指令对象
    if(actionObjects == "undefined" || actionObjects.length == 0) return;
    
    // 显示确认提示框（如果启用）
    if(!tipBox.showTip) {
        if(tipBox.visible && (tipBox.text == qsTr("Make sure to insert program?"))) return;
        tipBox.showCheck(
            qsTr("Make sure to insert program?"),
            qsTr("Yes"),
            qsTr("No"),
            qsTr("Do Not Show Again"),
            tipBox.showTip
        );
        tipBox.checkFinished.connect(tipBox.showProgramTip);
    } else {
        // 直接插入程序
        _InsertPrograms(actionObjects);
    }
}
```

function _InsertPrograms(obj) {
    // 遍历指令对象数组，逐个插入
    for(var i = 0; i < obj.length; ++i) {
        insertActionToList(obj[i]);
    }
    
    // 刷新程序列表显示
    repaintProgramItem(currentModel());
    
    // 定位到当前行
    programListView.positionViewAtIndex(programListView.currentIndex, ListView.Visible);
}
```

### 9. 单个指令插入

**文件**: `qml/App_HAMOUI/teach/ProgramFlowPage.qml` (第 200-256 行)

```qml
function insertActionToList(actionObject) {
    var copyrevokebuff = [];
    var id;
    
    // 获取当前选中的行索引
    var cI = programListView.currentIndex;
    var oCI = cI;
    if(cI < 0) return;
    
    // 设置插入索引
    actionObject.insertedIndex = PData.programToInsertIndex[currentProgramIndex()]++;
    
    var cPI = currentProgramIndex();
    var model = currentModel();
    
    // 同步特殊行信息（向后移动一行）
    PData.counterLinesInfo.syncLines(cPI, oCI, 1);  // 计数器行
    PData.timerLinesInfo.syncLines(cPI, oCI, 1);    // 定时器行
    PData.stackLinesInfo.syncLines(cPI, oCI, 1);    // 堆栈行
    PData.pointLinesInfo.syncLines(cPI, oCI, 1);    // 点位行
    
    // 处理标志定义
    if(actionObject.action === Teach.actions.ACT_FLAG) {
        Teach.currentRecord.flagsDefine.pushFlag(
            cPI, 
            new Teach.FlagItem(actionObject.flag, actionObject.comment)
        );
    }
    
    // 处理计数器ID
    if(Teach.hasCounterIDAction(actionObject)) {
        var cs = Teach.actionCounterIDs(actionObject);
        for(var c in cs) {
            PData.counterLinesInfo.add(cPI, cs[c], cI);
            id = cs[c];
        }
    }
    
    // 处理定时器ID
    if(Teach.hasTimerIDAction(actionObject)) {
        cs = Teach.actionTimerIDs(actionObject);
        for(c in cs) {
            PData.timerLinesInfo.add(cPI, cs[c], cI);
            id = cs[c];
        }
    }
    
    // 处理堆栈ID
    if(Teach.hasStackIDAction(actionObject)) {
        PData.stackLinesInfo.add(cPI, Teach.actionStackID(actionObject), cI);
        id = Teach.actionStackID(actionObject);
    }
    
    // 处理点位引用
    if(Teach.canActionUsePoint(actionObject)) {
        var points = Teach.currentRecord.definedPoints.parseActionPointsHelper(actionObject);
        for(var p = 0; p < points.length; ++p) {
            PData.pointLinesInfo.add(cPI, points[p].index, cI);
            id = points[p].index;
        }
    }
    
    // 插入到模型中（在当前行位置插入）
    model.insert(cI++, new Teach.ProgramModelItem(actionObject, Teach.actionTypes.kAT_Normal));
    
    // 标记程序已修改
    hasModify = true;
    
    // 记录操作日志
    var text = ICString.icStrformat(
        qsTr("OP:Insert  Edit:{0},Model:{1},Line:{2}Log:{3}"),
        editing.currentText(),
        moduleSel.currentText(),
        (programListView.currentIndex - 1) + "\n",
        actionObjectLogToText(actionObject)
    );
    ICOperationLog.opLog.appendOperationLog(text);
    
    // 保存到撤销缓冲区
    copyrevokebuff = revokebuff;
    copyrevokebuff.push({
        "programIndex": cPI,
        "edit": editing.currentText(),
        "currentmodule": moduleSel.currentText(),
        "id": id,
        "line": oCI,
        "actionObject": actionObject,
        "log": text
    });
    revokebuff = copyrevokebuff;
}
```

## 特殊处理逻辑

### 1. 轴掩码（mask）处理

轴掩码是一个二进制数，每一位表示对应轴是否有效：

```javascript
// 例如：mask = 63 = 0b00111111
// 表示 J1-J6 轴有效，J7-J8 轴无效

var j6Mask = ((mask >> 6) & 1);  // 检查第7轴是否有效
var j7Mask = ((mask >> 7) & 1);  // 检查第8轴是否有效

if(j6Mask) {
    obj.m6 = actionObject.m7;  // 添加第7轴坐标
}
if(j7Mask) {
    obj.m7 = actionObject.m8;  // 添加第8轴坐标
}
```

### 2. 点位数据提取

从 `points` 数组中提取坐标到 `point1-12` 变量：

```javascript
var info1 = points[0].pos;
for(var i in info1) {
    if(i == "m0") point1 = info1[i];  // X轴
    if(i == "m1") point2 = info1[i];  // Y轴
    if(i == "m2") point3 = info1[i];  // Z轴
    if(i == "m3") point4 = info1[i];  // A轴
    if(i == "m4") point5 = info1[i];  // B轴
    if(i == "m5") point6 = info1[i];  // C轴
    if(i == "m6") m7 = info1[i];      // 第7轴
    if(i == "m7") m8 = info1[i];      // 第8轴
}
```

### 3. 特殊行信息同步

当插入新指令时，需要同步所有特殊行的行号：

```javascript
// 在插入位置之后的所有特殊行，行号都要+1
PData.counterLinesInfo.syncLines(programIndex, insertLine, 1);
PData.timerLinesInfo.syncLines(programIndex, insertLine, 1);
PData.stackLinesInfo.syncLines(programIndex, insertLine, 1);
PData.pointLinesInfo.syncLines(programIndex, insertLine, 1);
```

### 4. 操作日志记录

每次插入操作都会记录详细的日志：

```javascript
var text = ICString.icStrformat(
    qsTr("OP:Insert  Edit:{0},Model:{1},Line:{2}Log:{3}"),
    editing.currentText(),      // 当前编辑程序名称
    moduleSel.currentText(),    // 当前模块名称
    programListView.currentIndex, // 插入行号
    actionObjectLogToText(actionObject) // 指令详细信息
);
ICOperationLog.opLog.appendOperationLog(text);
```

## 注意事项

1. **点位有效性检查**
   - 插入前必须确保点位已示教或定义
   - 点位坐标必须在机器人工作范围内

2. **参数合法性**
   - 速度范围：0-100%
   - 延时范围：0-999.9秒
   - 平滑度等级：0-9

3. **工艺类型切换**
   - 切换工艺类型后需要断电重启才能完全生效
   - 切换前建议保存当前程序

4. **程序修改标志**
   - 插入指令后 `hasModify` 标志会被设置为 `true`
   - 退出前需要保存程序，否则修改会丢失

5. **撤销功能**
   - 每次插入操作都会保存到撤销缓冲区
   - 可以通过撤销功能恢复到插入前的状态

6. **自动模式限制**
   - 在自动运行模式下，某些编辑功能会被禁用
   - 需要切换到手动模式才能进行程序编辑

## 总结

取片工艺教导插入流程是一个典型的分层架构设计：

整个流程通过 QML 的信号槽机制和 JavaScript 函数调用实现了松耦合的模块化设计，便于维护和扩展。
